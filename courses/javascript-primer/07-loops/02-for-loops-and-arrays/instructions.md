### Exercise

Write a function called `sumNumbers`. This function should:

  - return the sum of all numbers of the passed array
  - return 0 if the array has no elements

For example:

```js
sumNumbers([]);        // returns 0
sumNumbers([1, 2, 3]); // returns 6
```
